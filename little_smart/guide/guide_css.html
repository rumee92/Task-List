CSS 파일 정의

  • common.css : 태그 리셋 및 공통 스타일 정의
  • main.css : 메인 컨텐츠 스타일 정의
  • contents.css : 서브 컨텐츠 스타일 정의

주석표기

대분류 (Depth1)

/** 태그 초기화 **/

소분류 (Depth2)

/* [폰트색상] */

작성 규칙

선택자 마지막과 속성별로 띄어쓰기를 하지 않는다.

.inputType{display:inline-block;vertical-align:middle;}

문자열이 들어가는 경우 싱글쿼터('')로 감싸준다.

background:#fff url('/images/파일명') no-repeat left top / cover

작성 순서 (속성 우선 순위)

  • ⅰ. 블럭타입 : / display  /  overflow  /  position  /  left(right)  /  top(bottom)  /  z-index  /  float  /  clear
  • ⅱ. 레이아웃 : / width  /  height  /  padding  /  margin  /  border
  • ⅲ. 정렬 : / vertical-align  /  text-align  /  text-indent  /  text-overflow
  • ⅳ. 폰트 : / font-size  /  font-family  /  font-weight  /  line-height  /  color  /  letter-spacing  /  white-space  /  word-break  /  word-wrap
  • ⅴ. 디자인 : / background  /  background-size  /  outline  /  opacity  /  cursor
  • ⅵ. css3 : / border-radius  /  box-shadow  /  box-sizing  /  transition  /  transform  /  animation  /  gradient

CSS3속성 정의 방법

CSS3속성은 벤더프리픽스와 함께 한 줄로 작성하며, 속성별로 줄바꿈한다.
벤더프리픽스는 webkit, moz, ms, o 순서로 나열

-webkit-transition:all 3ms ease;-moz-transition:all 3ms ease;-ms-transition:all 3ms ease;-o-transition:all 3ms ease;transition:all 3ms ease; -webkit-transform:scale(1);-moz-transform:scale(1);-ms-transform:scale(1);-o-transform:scale(1);transform:scale(1);

복수 선택자 정의

복수 선택자를 선택하여 적용할 경우 ,(콤마)이후 줄바꿈으로 정렬한다.

.paging span.first, .paging span.prev, .paging span.next, .paging span.last

유의사항

  • 공통 클래스의 사용은 변경될 여지가 적을 때 최소한으로 사용하기를 권장
  • 레이아웃이 특정 페이지에서만 변경되는 예외 케이스가 생길 경우 div#wrap에 클래스를 추가하여 적용되도록 작업한다.
    div#wrap 외의 기본 레이아웃 div에는 별도의 클래스를 추가하지 않도록 한다.
